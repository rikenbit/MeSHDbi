# MeSHDbi

DBI to construct MeSHDb object from SQLite file.

Installation
======
~~~~
git clone https://github.com/rikenbit/MeSHDbi/
cd MeSHDbi
R CMD INSTALL MeSHDbi
~~~~
or type the code below in the R console window
~~~~
library(devtools)
devtools::install_github("rikenbit/MeSHDbi")
~~~~
